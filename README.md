# KEMRI AI Literature Assistant

An AI-powered literature review platform optimized for KEMRI research priorities (malaria, TB, genomics, outbreak surveillance).

## 🎯 Features
- **Natural Language Search**: Ask research questions in plain English
- **AI-Powered Summarization**: Instant summaries of multiple papers
- **Research Gap Analysis**: Identifies understudied areas
- **African Context Priority**: Prioritizes African research
- **Citation-Ready Exports**: Word, PowerPoint, and formatted citations
- **Bohrium-Style Interface**: Clean, professional two-panel layout

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
